Proyecto de Programación - Formulario de Droguerías

Descripción del Proyecto

Este proyecto tiene como objetivo desarrollar una aplicación en Java con interfaz gráfica para la gestión de datos de droguerías. La aplicación permite ingresar información mensual y realizar cálculos sobre las ventas. Además, se busca proporcionar a los usuarios una experiencia intuitiva y eficiente al interactuar con la interfaz de la aplicación.

Objetivos

  • Implementar interfaz gráfica en Java.
  • Realizar cálculos sobre las ventas mensuales.
  • Almacenar y cargar datos en archivos de texto y en la nube.
  • Implementar un mecanismo de detección y notificación de errores para informar a los usuarios sobre posibles errores de ingreso de datos.

Tecnologías Utilizadas

  • Java y Maven
  • HTML, CSS, JS (para el sitio web)
  • JSON para el almacenamiento estructurado de datos
  • Plataforma de almacenamiento en la nube (GitLab)

Beneficios Esperados

  • Aplicación práctica de conocimientos en desarrollo Java.
  • Integración de diversas tecnologías y validación de datos.
  • Mejora en habilidades de trabajo colaborativo y presentación de proyectos.
